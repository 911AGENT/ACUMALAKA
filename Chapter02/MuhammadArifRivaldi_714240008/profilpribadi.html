About Me :

Saat ini saya sedang belajar Python dan JavaScript. Saya terpesona dengan bagaimana bahasa-bahasa ini dapat digunakan untuk membuat segala sesuatu mulai dari skrip sederhana hingga aplikasi web yang kompleks. Saya senang menjelajahi dasar-dasar HTML, CSS, dan JavaScript untuk membangun situs web yang interaktif dan menarik secara visual. Saya menyukai tantangan dan teka-teki, dan coding menawarkan platform yang sempurna untuk menguji kemampuan pemecahan masalah saya. Saya bertujuan untuk mempelajari lebih banyak bahasa pemrograman dan kerangka kerja untuk memperluas keahlian saya. Saya juga ingin mengerjakan proyek dunia nyata yang dapat membuat perbedaan dan memecahkan masalah praktis. Saya ingin terhubung dengan pelajar dan profesional lain di bidangnya untuk berbagi pengetahuan dan berkolaborasi dalam proyek menarik. Saya selalu terbuka terhadap peluang dan koneksi baru. Kritik dan saran sangat diperlukan guna menilai kemampuan untuk bisa berkembang di kemudian hari. Terima kasih telah mengunjungi halaman saya.
